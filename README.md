# 🏡 Dream Home AI Chatbot

An AI-powered real estate chatbot developed during my AI Engineering Internship at **TechNexus**. This project is designed to assist users with property-related inquiries through natural language conversations using the Groq API and Python.

## 👨‍💻 About the Project

The Dream Home AI Chatbot provides an interactive experience for users looking for real estate information. It leverages Large Language Models (LLMs) through the Groq API to deliver intelligent and conversational responses.

This project serves as the foundation for a smart real estate assistant and is designed to support future enhancements such as property search, AI-powered recommendations, conversation memory, and appointment scheduling.

## ✨ Features

- 🤖 AI-powered conversational chatbot
- ⚡ Real-time streaming responses using Groq API
- 🏡 Answers real estate and property-related questions
- 💬 Interactive command-line interface
- 🔒 Secure API key management using environment variables (recommended)
- 📈 Designed for future integration with property datasets and AI recommendation systems

## 🛠️ Technologies Used

- Python
- Groq API
- Llama 3.1
- JSON
- Visual Studio Code

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/dream-home-ai-chatbot.git
cd dream-home-ai-chatbot
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Configure your API key

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

### Run the chatbot

```bash
python chatbot.py
```
